This is the API for the MakStat-Revised project. If you want to know what this project is about, pleace check this document (accessible from any SEEU account).

To see our data visualized, please go to this page. In the current page, you can access our data in JSON format at the following endpoints (with the provided URI formats):

    Data types:

    • Categories and sub-categories for each category
      • /categories/{category}/subCategories/{subCategory}

    Data:

    • Employee counts
      • /employeeCount/{category}/{subCategory}/{year}/{gender}
    • Average net wages
      • /averageWage/{category}/{subCategory}/{year}/{gender}
    • Unemployment rates
      • /unemploymentRate/{category}/{subCategory}/{year}/{gender}
    • Wage distribution
      • /wageDistribution/{category}/{subCategory}/{year}/{wageGroup}
